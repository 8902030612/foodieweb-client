# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Installation

Install with npm or yarn or pnpm

```bash
  npm install 

  yarn install   

  pnpm i   
  
```
